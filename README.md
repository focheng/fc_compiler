# FC Compiler
 
## Folder Structure
- common - Common implementation
  - lexer
  - token
  - parser
  - ast
  - il - intermediate language
  - asm
  - linker
- language - Language specific implementation
  - c
  - java
  - cobol
    - netcobol
  - silktest
- runtime - Runtime libraries
  - cobol
  - silktest
- app - Applications
  - interpreter
  - translator
